In the box

  • mix - command line
  • iex - repl
  • ecto - database mapper / migrations
  • mnesia - internal distributed database / cache
  • process supervisor for running distributed systems
  • logging - OpenTelemetry

Interactive


						iex(1)> h IO.puts/1

						def puts(device \\ :stdio, item)

						Writes item to the given device, similar to write/2, but adds  
						a newline at the end.

						By default, the device is the standard output. It returns :ok 
						if it succeeds.

						## Examples

							IO.puts("Hello World!")
							#=> Hello World!

							IO.puts(:stderr, "error")
							#=> error
